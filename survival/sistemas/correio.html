📬Correio

Pru.. pru.. chegou o "pombo correio" para te auxiliar no envio ou recebimento de itens!

Este sistema oferece aos jogadores a possibilidade de enviar e receber itens, seja através de transações com outros jogadores ou por meio de envios diretos do servidor, tais como as doações efetuadas no site, recompensas de passe e torneios.

Como verificar o meu correio?

Para visualizar o seu correio, você pode usar o comando /correio (o que abrirá uma janela mostrando o menu do sistema). Neste menu você consegue visualizar as encomendas e recompensas que tem armazenadas.

Como recolher o meu correio?

Para recolher os itens armazenados, primeiro precisa selecionar a categoria que pretende recolher. Para alternar entre as 3 categorias clique no funil.

Ao selecionar a categoria desejada clique em recolher todos, para receber os itens disponiveis.

Como posso enviar uma encomenda?

Dentro do menu sistema (/correios), selecione a opção “Enviar” para assim enviar uma encomenda a outros jogadores.

Posteriormente, irá se abrir um novo menu onde deve inserir os itens que pretende enviar.

Por fim, com a transação confirmada irá receber uma notificação do chat do servidor informando o envio da encomenda.
